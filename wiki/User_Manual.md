# Guía de Trading — El Séquito del Terror

## Estrategias de Mercado

### 1. Escaneo Matutino
Realiza un /aux scan completo al inicio de cada sesión. Esto actualiza tu base de datos de precios y te permite identificar oportunidades de arbitraje.

### 2. Materiales de Crafting
Monitor prioritario de:
- **Fórmulas de encantamiento raras** (alto margen, baja oferta)
- **Materiales de Farm masivo** (pociones de maná, vendas, reactivos de raid)
- **Minerales y hierbas de nivel 50-60** (demanda constante en Turtle WoW)

### 3. Umbral de Compra
Configura alertas en AUX para comprar automáticamente cuando el precio esté:
- **Por debajo del 70%** del precio medio → comprar para reventa
- **Por encima del 130%** del precio medio → momento ideal para vender

### 4. Financiación del Clan
El Séquito utiliza AUX para financiar:
- Pociones de raid (resistencia, élixires, comida de buff)
- Materiales de encantamiento para gear de nuevos miembros
- Reparaciones de equipo post-wipe

### Tips Avanzados
- Escanea la AH en horario de baja actividad (mañana entre semana) para encontrar mejores precios.
- Usa el historial de 7 días para identificar tendencias antes de invertir.
- Coordina con el banco del clan (WCS_Brain → Banco) para compras grupales.
