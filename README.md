## 💰 ¿Qué es AUX-Trading?

AUX-Trading reemplaza e mejora la interfaz de la Casa de Subastas de Turtle WoW con funciones avanzadas de búsqueda, análisis de precios históricos y automatización de pujas.

## 🚀 Características

| Función | Descripción |
|---|---|
| **Búsqueda Avanzada** | Filtros por tipo, nivel, calidad, precio mínimo/máximo |
| **Historial de Precios** | Base de datos local de precios escaneados |
| **Comparación** | Compara con el precio medio del mercado |
| **Alertas** | Notificación cuando aparece un item por debajo del umbral |
| **Buyout Rápido** | Compra instantánea con un clic |
| **Resell Analysis** | Calcula el margen potencial de reventa |

## ⚙️ Instalación

1. Extrae en Interface\AddOns\aux-addon\.
2. Activa el addon.
3. Abre la Casa de Subastas en el juego — AUX reemplaza automáticamente la UI.

## 💻 Comandos

| Comando | Descripción |
|---|---|
| /aux | Abrir configuración |
| /aux scan | Escanear toda la casa de subastas |
| /aux reset | Resetear base de datos de precios |
