# aux-addon — v9.3.0 [God-Tier] ⚔️🛡️

Mejora masiva para la casa de subastas. Domina el mercado de Turtle WoW.

## 🚀 Características
- Optimización extrema para Turtle WoW.
- Integración nativa con el **WCS_Brain**.
- Interfaz modernizada y limpia.

## ⚙️ Instalación
1. Descarga el ZIP.
2. Extrae en Interface\AddOns\aux-addon.
3. Reinicia el juego.
